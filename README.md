# AutoClicker for Windows

## Installation & Usage

### Easiest Method (Recommended)
1. Make sure you have **Python 3.7 or higher** installed
   - Download from: https://www.python.org/downloads/
   - ✅ Check "Add Python to PATH" during installation

2. **Double-click** `build_windows.bat`
   - This will build `AutoClicker.exe`
   - Wait for it to finish (may take 2-3 minutes)

3. When done, find `AutoClicker.exe` in the `dist` folder
   - Double-click to run!

### Manual Method
```cmd
pip install -r requirements.txt
python autoclicker.py
```

## Controls
- **F6** - Start/Stop clicking
- **F7** - Record position
- **F8** - Clear all positions
- Or use the GUI buttons

## Features
- Record multiple click positions
- Adjustable click speed
- Left, right, or middle mouse button
- Works standalone once built!

## Troubleshooting

**"Python not found"?**
- Install Python: https://www.python.org/downloads/
- Make sure to check "Add Python to PATH"

**Build fails?**
- Right-click `build_windows.bat` → Run as Administrator

**Clicking doesn't work?**
- Try running as Administrator
- Check if the app has permissions in Windows Defender

Enjoy! 🎉
